# PVE-CPatch / Proxmox VE 正體中文化更新檔


## 使用說明
  
直接把「pve-lang-zh_CN.js」覆蓋到 /usr/share/pve-manager/locale/ 內同名檔案即可。
  
這個版本很粗淺，會利用時間繼續完善。
